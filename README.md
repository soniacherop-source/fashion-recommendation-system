# Fashion Recommendation System Using Collaborative Filtering

## Project Overview

This project is a machine learning-based fashion recommendation system that suggests fashion products to users using collaborative filtering techniques. The system analyzes user-product interactions and recommends products based on similarities between users.

The project demonstrates the complete machine learning workflow including:
- Business Understanding
- Data Understanding
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Collaborative Filtering Modeling
- Recommendation Generation
- Model Evaluation and Visualization

---

## Dataset Features

The dataset contains:
- User ID
- Product ID
- Product Name
- Brand
- Category
- Price
- Rating
- Color
- Size

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Technique

The recommendation system uses:
- User-Based Collaborative Filtering
- Cosine Similarity
